# Changelog

## 1.2.0
- Added 1 Player Twinstick Functionality.
- Twinstick must be selected first before 1P Twinstick can be toggled on.
- 1p Twinstick must be toggled back off to be able to switch to 2 Player or 4 Player mode.
- Added GPIO 23 for 1P Twinstick on (Back button on Mid Left connector on Pacto 4000T).
- Added GPIO 24 for De-activate 1P Twinstick and go back to 2P Twinstick (Back button on Far Left connector on Pacto 4000T).
- This is how the Pacto firmware was designed when adding single Player Twinstick functionality.
